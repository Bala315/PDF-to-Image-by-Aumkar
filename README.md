# PDF-to-Image-by-Aumkar
Using imagick &amp; core PHP

**How To Use:**

First Install imagick:

```
sudo apt-get install php7-imagick
```

Then Create the upload directory and inside it pdf & image folder. Tree View:

-upload
    
    -image
    
    -pdf
